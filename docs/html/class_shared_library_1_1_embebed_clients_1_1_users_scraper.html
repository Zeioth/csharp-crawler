void SharedLibrary.EmbebedClients.UsersScraper.GetUsersData ( List< string >  userUrls)

This is an embebed client. It recieves a list of user urls. Then makes a request using firefox, parse the html code, and store every user and their apps into DB.

Please note that this method only return users with at least one app that already exists in appsdata collection and has a category in Constants.VALID_CATEGORIES.

You can get userUrls from the return of ReviewsCrawler.
