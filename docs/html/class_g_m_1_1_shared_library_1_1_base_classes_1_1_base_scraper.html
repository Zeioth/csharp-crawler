Performs a request to playstore and get The HTML code of this app page.

Controlled exception: -Retry and notify on request error. -After n request errors, delete from queue and try the next app. -Unless we are banned, then retry forever. -On too many failed requests send an email (global counter).

Returns
The HTML code of this app page
