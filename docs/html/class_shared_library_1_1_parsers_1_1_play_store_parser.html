AppModel SharedLibrary.Parsers.PlayStoreParser.ParseAppPage ( string  response,
string  pageUrl 
)

Parses ALL data from this app's page.

Controlled errors: -If an error stops the program, no more apps will be added or updated until this parser is manually repaired. This way our information will never be corrupted.

Parameters
responseHTML Response of the App's landing page
pageUrlURL of the App's landing page
Returns
Parsed app data structure. It returns null as flag to skip this app.
