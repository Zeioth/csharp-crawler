)

Performs a request to playstore and get The HTML code of this app page.

503 error means we are banned, or the application is not available for our country. Controlled exception: -Retry and notify on request error. -After n request errors, delete from queue and try the next app. -Unless we are banned, then retry forever. -On too many failed requests send an email (global counter).

Returns
The HTML code of this app page
Parameters
appIdApp identifier.
cleanerModeIf set to true, if an app doesn't exists anymore, delete it from appsdata.
