List<string> SharedLibrary.EmbebedClients.ReviewsCrawler.CrawlReviews ( AppModel  parsedApp,
int  parsedUsers = -1 
)

This is an embebed client. It crawls all the reviews from an app for every languaje configured in constants. It will get the html code, parse it, and store the results into db.

Because we need to parse a review in order to parse his users, if parseReviews is disabled but parseUsers is enabled, we'll only parse the reviews necesary to parse the users, and they will be erased afterwards.

Returns
A list of reviews authors.
Parameters
parsedAppParsed app.
parsedUsersNumber of users already parsed by the UsersWrapper. If only –users is enabled, and max users have been parsed. Then this method will not execute anymore. (This is an optimization).
